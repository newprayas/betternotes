# Image Upload Guidelines for Best Display

## Recommended Image Dimensions

### Portrait Images
- **Optimal size**: 800 × 1000 pixels (4:5 aspect ratio)
- **Alternative**: 864 × 1152 pixels (3:4 aspect ratio)
- **Maximum quality**: 900 × 1200 pixels (3:4 aspect ratio)

### Landscape Images
- **Optimal size**: 1920 × 1080 pixels (16:9 aspect ratio)
- **Alternative**: 1600 × 1200 pixels (4:3 aspect ratio)
- **Square option**: 1080 × 1080 pixels (1:1 aspect ratio)

## File Specifications

### File Format
- **Preferred**: WebP (for better compression and quality)
- **Acceptable**: JPEG, PNG
- **Avoid**: GIF, BMP (large file sizes)

### File Size
- **Maximum**: 5MB per image
- **Optimal**: Under 2MB for faster loading
- **Recommended**: Compress images before uploading

### Color Profile
- **RGB**: For web display
- **Avoid**: CMYK (for print only)

## Quality Settings

### JPEG Compression
- **Quality**: 80-90%
- **Progressive**: Enabled
- **Optimized**: For web

### PNG Settings
- **Compression**: Lossless
- **Colors**: 24-bit for photos, 8-bit for graphics

## Mobile Optimization

### Responsive Considerations
- Images will automatically resize for different devices
- Portrait images work best on mobile devices
- Landscape images display well on desktop and tablet

### Aspect Ratio Preservation
- The slideshow maintains the original aspect ratio of your images
- No cropping or zooming is applied
- Black bars are eliminated by matching container to image dimensions

## Common Issues and Solutions

### Black Bars on Sides
- **Cause**: Mismatch between container and image aspect ratio
- **Solution**: Upload images with consistent aspect ratios
- **Fixed**: Our slideshow now adapts to your image's exact dimensions

### Slow Loading
- **Cause**: Large file sizes or uncompressed images
- **Solution**: Optimize images before uploading
- **Tools**: TinyPNG, ImageOptim, or online compressors

### Poor Quality
- **Cause**: Low resolution images or excessive compression
- **Solution**: Use recommended dimensions and appropriate quality settings

## Upload Workflow

1. **Resize** your image to the recommended dimensions
2. **Compress** the image to reduce file size
3. **Convert** to WebP format if possible
4. **Upload** through the Sanity studio
5. **Preview** in the slideshow to verify display

## Tools for Image Optimization

### Online Tools
- TinyPNG (tinypng.com)
- Squoosh (squoosh.app)
- ImageCompressor (imagecompressor.com)

### Desktop Software
- Adobe Photoshop (Save for Web)
- GIMP (Free alternative)
- ImageOptim (Mac)

### Command Line Tools
- ImageMagick
- Sharp (Node.js)
- cwebp (WebP converter)

## Testing Your Images

After uploading, check:
- Image displays without black bars
- Loading time is acceptable
- Quality is maintained on different devices
- No distortion or stretching occurs

## Support

For issues with image display:
1. Verify your image meets the recommended specifications
2. Check file size is under 5MB
3. Ensure format is supported (JPEG, PNG, WebP)
4. Contact support if issues persist
